# tile_wm

Windows 11 用のカスタムトップタスクバーアプリケーションです。仮想デスクトップの切り替えとウィンドウタイリングをシンプルな UI で提供します。

## 機能

### 仮想デスクトップ切り替え
- タスクバー上のボタンで仮想デスクトップをワンクリック切り替え
- `Ctrl+Win+←/→` などのシステムショートカットによる切り替えにもリアルタイムで追従

### ウィンドウタイリング
アクティブなウィンドウを基準に、以下の 5 つのレイアウトモードを切り替えできます：

| モード | レイアウト |
|--------|-----------|
| Free | タイリングなし（自由配置） |
| 2 windows | （左メイン＋右） <--> （右メイン＋左） |
| 3 windows | （左メイン＋右上＋右下） <--> （右メイン＋左上＋左下）|
| 4 windows | （左上メイン＋右上＋左下＋右下） <--> （右上メイン＋左上＋右下＋左下） |

※ メインウィンドウの下には余分なウィンドウが自動的にスタックされます。

### タスクバー配置
- **フロート** (Float)  メニューボタンをドラッグして自由に移動可能。

### その他
- 画面最上部に常時表示されるトップバーとして動作
- Windows AppBar として登録し、他のウィンドウの作業領域を自動的に調整
- 透明背景でデスクトップに自然に溶け込むデザイン
- アプリアイコンをクリックするとメニューを表示（バージョン情報、設定ファイル編集、ヘルプ、終了）

## 必要要件

- **OS**: Windows 11
- **Rust**: 2021 Edition 以降
- [Cargo](https://doc.rust-lang.org/cargo/)

## ビルド

```bash
cargo build --release
```

実行ファイルは `target/release/tile_wm.exe` に生成されます。

## 実行

```bash
cargo run --release
```

または、ビルド済みの `tile_wm.exe` を直接実行してください。

## 設定

設定ファイルは以下のパスに自動生成されます：

```
%LOCALAPPDATA%\tile_wm\config.toml
```

### 設定項目

| キー | 型 | デフォルト | 説明 |
|------|-----|-----------|------|
| `bar_height` | 整数 | `40` | タスクバーの高さ（ピクセル） |
| `split_ratio_x` | 整数 | `50` | 画面の水平分割位置（%） |
| `split_ratio_y` | 整数 | `50` | 画面の垂直分割位置（%） |
| `exclude_titles` | 配列 | `[]` | タイリングから除外するウィンドウタイトルの部分一致リスト |
| `exclude_processes` | 配列 | `[]` | タイリングから除外するプロセス名のリスト |
| `float_x` | 浮動小数点 | `100.0` | フロートモード時のX座標 |
| `float_y` | 浮動小数点 | `100.0` | フロートモード時のY座標 |
| `float_width` | 浮動小数点 | `400.0` | フロートモード時の幅 |
| `float_bg_rgba` | 配列 | `[32, 32, 32, 255]` | フロートモード時の背景色 (RGBA) |

### 設定例

```toml
bar_height = 36
split_ratio_x = 50
split_ratio_y = 50
exclude_titles = ["Calculator"]
exclude_processes = ["tile_wm.exe"]
```

設定ファイルはメニューの **Edit config.toml** ボタンからも開くことができます。
